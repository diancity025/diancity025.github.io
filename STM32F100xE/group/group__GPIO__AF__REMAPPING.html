__STATIC_INLINE void LL_GPIO_AF_DisableRemap_MISC ( void  )

Miscellaneous features remapping.

This bit is set and cleared by software. It controls miscellaneous features. The DMA2 channel 5 interrupt position in the vector table. The timer selection for DAC trigger 3 (TSEL[2:0] = 011, for more details refer to the DAC_CR register).

Reference Manual to LL API cross reference:
MAPR2 MISC_REMAP LL_GPIO_AF_DisableRemap_MISC
Note:
DISABLE: DMA2 channel 5 interrupt is mapped with DMA2 channel 4 at position 59, TIM5 TRGO event is selected as DAC Trigger 3, TIM5 triggers TIM1/3.
This bit is available only in high density value line devices.
Return values:
None
